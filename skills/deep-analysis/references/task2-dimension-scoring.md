# Task 2 · 19 维打分

读取 `raw_data.json`，对每个维度按规则打 1-10 分，写入 `dimensions.json`。

## 通用打分公式

每个维度产出：
```json
{
  "score": 7,            // 1-10 整数
  "weight": 5,           // 1-5 星权重
  "label": "趋势向上",    // 一句话标签
  "reasons_pass": ["ROE 连续 3 年 > 18%", "净利率 22%"],
  "reasons_fail": ["商誉占净资产 35%"],
  "raw_pointer": "raw_data.json#/dimensions/1_financials"
}
```

## 19 维评分细则

### 维度 1 · 财报（权重 5）
```
9-10: ROE>20% 连续 5 年 + 营收增速>30% + 净利率>20% + 现金流优秀
7-8:  ROE>15% 连续 3 年 + 营收增速>20% + 净利率>15%
5-6:  ROE>10% + 营收增速>10% + 净利率>10%
3-4:  ROE 5-10% + 营收增速 <10%
1-2:  ROE<5% / 净利下滑 / 现金流为负
```

### 维度 2 · K 线（权重 4）
```
9-10: Stage 2 初期 + 均线多头 + 放量突破 + MACD 水上金叉
7-8:  趋势向上 + 均线支撑有效 + 量价配合
5-6:  横盘整理 + 方向不明
3-4:  趋势偏弱 + 均线空头
1-2:  Stage 3/4 + 破位 + 放量下跌
```

### 维度 3 · 宏观（权重 3）
- 利率周期对该行业利好/利空 (±1)
- 汇率方向对营收影响 (±1)
- 地缘风险敞口 (±1)
- 大宗商品周期 (±1)
- 起评分 6 分

### 维度 4 · 竞争对手（权重 4）
- 市场份额排名第 1 → +3，第 2-3 → +2，第 4-5 → +1
- 毛利率高于行业均值 → +1
- 营收增速高于头部竞争者 → +2
- 估值低于行业均值 → +1
- 起评分 3

### 维度 5 · 上下游（权重 4）
- 议价能力强（上下游分散）→ +3
- 客户/供应商集中度 < 30% → +2
- 在产业链微笑曲线两端 → +2
- 起评分 3

### 维度 6 · 研报（权重 3）
- 近 3 月覆盖券商 ≥ 10 → +1
- 一致预期目标价相比现价 +20% 以上 → +3
- 买入/增持评级占比 > 70% → +2
- 重点券商均看多 → +1
- 起评分 3

### 维度 7 · 行业景气（权重 4）
```
9-10: 增速>30% + 渗透率<20% + 强催化
7-8:  增速 15-30% + 渗透率提升中
5-6:  增速 5-15% + 成熟期
3-4:  增速<5%
1-2:  萎缩
```

### 维度 8 · 原材料（权重 3）
- 核心原材料价格下行 → +3
- 成本占比 < 30% → +1
- 进口依赖度 < 20% → +1
- 公司有对冲能力 → +1
- 起评分 4

### 维度 9 · 期货关联（权重 2）
- 关联期货品种价格趋势利好 → 7-9
- 中性 → 5-6
- 利空 → 1-4
- 无关联 → 5

### 维度 10 · 估值（权重 5）
```
9-10: 多模型交叉验证严重低估 (>40% 上涨空间)
7-8:  明显低估 (20-40%)
5-6:  合理估值 (±20%)
3-4:  偏贵 (高 20-40%)
1-2:  严重高估 (>40%)
```
必须报告 PE/PB/PEG/历史分位 4 个数字。

### 维度 11 · 治理（权重 4）
- 实控人质押 < 20% → +2
- 管理层近 12 月净增持 → +3
- 无重大违规 → +1
- 关联交易占比 < 5% → +2
- 股权激励有挑战性 → +1
- 起评分 1

### 维度 12 · 资金面（权重 4）
- 北向/南向资金 20 日净买入 → +3
- 融资余额上升 → +1
- 股东户数 3 季度持续下降（筹码集中）→ +3
- 主力资金近 5 日净流入 → +2
- 起评分 1

### 维度 13 · 政策（权重 3）
- 政策方向利好 → 8-10
- 中性 → 5-7
- 监管收紧 → 1-4

### 维度 14 · 专利护城河（权重 3）
- 巴菲特 5 类护城河至少占 2 类 → 7-10
- 占 1 类 → 5-6
- 无明显护城河 → 1-4

### 维度 15 · 事件驱动（权重 4）
- 近 30 天有强催化（订单/获批/并购）→ +3
- 业绩预告超预期 → +3
- 无重大利空 → +2
- 起评分 2

### 维度 16 · 龙虎榜（权重 4）
- 近 30 天有殿堂级游资上榜 → +3
- 机构买入金额 > 卖出 → +2
- 同板块龙头辨识度高 → +2
- 起评分 3

### 维度 17 · 舆情（权重 3）
- 大 V 提及级别 S → +3, A → +2, B → +1
- 讨论环比上升但非异常 → +2
- 情绪正面占比 > 50% → +2
- 起评分 1

### 维度 18 · 杀猪盘（权重 5，安全维度）
**反向计分**：信号越多分越低
- 0-1 信号 → 9-10 🟢
- 2-3 信号 → 6-8 🟡
- 4-5 信号 → 3-5 🟠
- 6+ 信号 → 1-2 🔴

### 维度 19 · 实盘比赛（权重 4）
- 淘股吧 S 级选手持有 ≥ 3 → +3
- 雪球高收益组合持有 ≥ 5 → +2
- A 级选手持有 → +2
- 起评分 3

## dimensions.json 结构

```json
{
  "ticker": "002273.SZ",
  "fundamental_score": 78.4,    // 加权平均 × 10
  "dimensions": {
    "1_financials": { "score": 8, "weight": 5, "label": "...", ... },
    "2_kline":      { "score": 7, "weight": 4, ... },
    "...": "..."
  }
}
```

加权平均公式：`sum(score_i * weight_i) / sum(weight_i) * 10`
