## 评分简化（适用于全部 22 位游资）

```
1. 用 is_in_range() 预过滤 → 不在射程直接 verdict="不适合"
2. 在射程则按以下加权:
   K线 (Stage / 量能 / 突破)         ×30
   龙虎榜匹配 (该游资席位是否上榜)    ×30
   情绪 (热度 / 大V / 板块温度)       ×20
   事件 (近期催化 / 公告)             ×20
3. 综合 ≥ 80 → 强烈买入  60-79 → 买入  <60 → 关注
```
